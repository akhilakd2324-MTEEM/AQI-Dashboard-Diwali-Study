# India AQI Dashboard (Dash + Plotly)

This is an interactive Air Quality Index (AQI) dashboard built with Python Dash.
It loads daily AQI data from `merged_aqi.csv` and visualizes:

- AQI gauge meter
- Time-series trend chart
- City selector dropdown
- Date slider

This project is fully deployable on Render.com for free.
